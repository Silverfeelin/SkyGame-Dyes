Sky Dye Tracker

Welcome to the Sky Dye Tracker! This website functions as a real-time interactive map to find and share dye plants in the game Sky: Children of the Light.
This website is experimental! Expect things to break.

To use the Sky Dye Tracker, please first authenticate through Discord. We won't ask for more than your username!

Source code is available on GitHub: Silverfeelin/SkyGame-Dyes.
