Privacy Policy

Last Updated: January 29, 2025
Effective: January 29, 2025

Welcome to the Sky Dye Tracker (the "Website"). We value your privacy and aim to be transparent about the data we collect, how we use it, and your rights regarding that information. This privacy policy outlines our practices concerning the collection, use, and disclosure of information when you use our Website.

All code for the Website, both client-side and server-side, is open-source and can be reviewed on the official Sky Dye Tracker GitHub repository.

1. Information We Collect

When you use the Website, you are required to authenticate through Discord. This is done to prevent misuse of the platform. As part of this process, the Website collects your Discord username and user ID.

2. Data Storage and Security

When you submit data to the Website, it is stored together with your Discord user ID and username in a Cloudflare account using technologies such as Durable Objects, D1, R2, and KV.
The submitted data is public and can be reviewed by other users. For more information, please refer to Cloudflare's storage options: Cloudflare Storage Options.

3. Third-Party Services

The Website uses Cloudflare Web Analytics to gain insights into website usage. This service does not track individual users and only provides aggregate data about page visits to help prioritize development. For more information, please refer to Cloudflare's web analytics privacy information: Cloudflare Web Analytics.

4. Cookies and Tracking Technologies

The Website uses a cookie to remember you when you authenticate through Discord. You can instruct your browser to refuse all cookies. However, if you do not accept cookies, you may not be able to use some portions of the Website.

5. User Rights

You have the right to access, update, or delete your data. You can do this by contacting us through the contact information provided below.

6. Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page. You are advised to review this privacy policy periodically for any updates. Changes to this privacy policy are effective as of the date at the top of the page.

7. No Affiliation with thatgamecompany

The Website is an independent, fan-made project and is not affiliated with, endorsed by, or sponsored by thatgamecompany. All trademarks, game assets, and intellectual property related to Sky: Children of the Light belong to thatgamecompany. This website is intended for informational purposes only and falls under fair use, as it aims to support the community by providing tools and resources without commercial intent.
